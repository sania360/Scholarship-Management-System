# 🎓 Scholarship Management System

A web-based Scholarship Management System developed using **PHP**, **MySQL**, **HTML**, and **CSS**. This project allows users to register students, manage scholarship records, and perform CRUD (Create, Read, Update, Delete) operations efficiently.

---

## 📌 Features

### 👨‍🎓 Student Module
- Student Registration
- Store student personal information
- Store educational details
- Save data into MySQL database

### 🎓 Scholarship Module
- Add new scholarships
- View available scholarships
- Update scholarship information
- Delete scholarships
- Display scholarship list in a table

## ⚙ Installation

### Step 1

Install **XAMPP**.

### Step 2

Copy the project folder into:

```
C:\xampp\htdocs\
```

### Step 3

Start

- Apache
- MySQL

from the XAMPP Control Panel.

### Step 4

Open **phpMyAdmin** and create a database named:

```
scholarship_system
```

### Step 5

Create the required tables or import the SQL file.

### Step 6

Open your browser and visit:

```
http://localhost/Scholarship-Management-System/
```
